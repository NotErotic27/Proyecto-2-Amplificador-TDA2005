# Proyecto 2 – Amplificador TDA2005

## Contenido del repositorio

- **Materiales proyecto 2.pdf**  
  Lista de componentes necesarios para la implementación del circuito.

- **Etapa drenador comun.pdsprj**  
  Archivo del diseño/simulación en Proteus de la etapa de drenador común.

- **Esquematico - proyecto 2.jpeg**  
  Diagrama esquemático del circuito implementado.

- **Diapositivas.pdf**  
  Presentación del proyecto utilizada para la exposición.

---

## Descripción del proyecto

Este proyecto consiste en el diseño y análisis de un amplificador TD, incluyendo:

- Selección de componentes  
- Diseño del circuito  
- Simulación  
- Implementación de la etapa de drenador común  
---

## Herramientas utilizadas

- Proteus (simulación)  
- Software de edición de documentos (Word / PDF)
